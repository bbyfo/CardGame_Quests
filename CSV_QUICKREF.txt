## CSV Column Format

```
Deck | CardName | TypeTags | AspectTags | InstructionType | InstructionSubType | InstructionTarget | InstructionTags | TargetRequirement
```

### Simple Example

```csv
Deck,CardName,TypeTags,AspectTags,InstructionType,InstructionSubType,InstructionTarget,InstructionTags,TargetRequirement
Verb,Defend,Protective,Military,,,,Evil Monster
Target,Raider,Evil;Monster,Military,Modify,Add,ThisCard,Hostile,

Twist,Ambush,Combat,Military,,,,

```

## Key Points

‚úì **Tags use semicolons**: `Evil Monster;Humanoid`
‚úì **Verbs need TargetRequirement**: What targets qualify
‚úì **Modify adds tags**: To ThisCard, Target, Location, Twist, Reward, or Failure
‚úì **Empty cells okay**: Most columns optional except Deck and CardName
‚úì **Deck values**: Verb, Target, Location, Twist, Reward, Failure (exact spelling!)

## Common Issues & Fixes

| Issue | Fix |
|-------|-----|
| "Invalid deck" | Check spelling: Verb, Target, Location, Twist, Reward, Failure |
| "Duplicate card name" | Each card in a deck must have unique name |
| "Missing TargetRequirement" | All Verbs need a TargetRequirement |
| Tags not working | Use semicolons: `Tag1;Tag2` not `Tag1,Tag2` |
| Import shows warnings | Usually okay to proceed, but check log |
