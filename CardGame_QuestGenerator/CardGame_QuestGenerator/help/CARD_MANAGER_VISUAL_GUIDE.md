## Responsive Behavior

### Desktop (> 1200px)
- Two column layout
- Form on left, Browser on right
- Full width inputs

### Tablet (768px - 1200px)
- Still two columns but narrower
- Slightly smaller padding
- Responsive grid adjusts

### Mobile (< 768px)
- Single column layout
- Form stacks on top of browser
- Full width buttons
- Touch-friendly spacing
- Scrollable lists
